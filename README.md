# 🐍 Snake Game

Um clássico jogo da cobrinha desenvolvido com **HTML5**, **CSS3** e **JavaScript Vanilla**, utilizando a API Canvas para renderização gráfica.

---

## 🎮 Sobre o Projeto

O Snake Game é um dos jogos mais conhecidos da história dos videogames.

Neste projeto, o jogador controla uma cobra que deve se alimentar para crescer e acumular pontos, evitando colisões com as paredes e com o próprio corpo.

O jogo foi desenvolvido sem frameworks ou bibliotecas externas, demonstrando conceitos fundamentais de desenvolvimento web e lógica de programação.

## 🐍 Mecânicas do Jogo

### Crescimento da Cobra

Quando a cobra come a comida:

* A pontuação aumenta
* Um novo alimento é gerado
* A cobra cresce

### Game Over

O jogo termina quando:

❌ A cobra colide com a parede

❌ A cobra colide com o próprio corpo

## 💾 Sistema de Recorde

O recorde é salvo automaticamente utilizando:

